# IPRO Home Add-on Repository (Main Branch)

This branch contains the **IPRO-12 Surgard Receiver** add-on files used by Home Assistant Supervisor
after the repository is registered via GitHub Pages.

## Structure

```
ipro-home-addon/
 ├── README.md
 └── ipro12_surgard_receiver/
       ├── config.json
       ├── Dockerfile
       ├── run.sh

       └── README.md
```
